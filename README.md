## Legends

### Status

| Icon | Status | Definition |
| --- | --- | --- |
| 🟩  | **Clear** | The mod has not caused any issue *(In the current version)*.<br/><br/>*This does not mean it won't cause issues as more mods are added or updates of the mod are pushed through and added.* |
| 🟥  | **Depracated** | The mod has been Removed/Disabled due to an issue that arose when installed<br/><br/>*This may have occured during the initial addition of the mod in the modpack, a sudden incompatability or issue upon the update of the mod or other mods within the modpack, or a sudden incompatability upon addition of a mod later in the modpack's lifespan* |
| 🟦  | **Untested** | The mod has been added to the modpack without running and opening a test world.<br/><br/>*In short, there is no quick verification to see wether the mod will work along side the other mods that are in the modpack* |
| 🟫  | **Planning** | The mod is planned to be added in the modpack.<br/><br/>*These are the backlogs of mods that are already planned to be added (but these mods are not safe from being replaced by better alternatives or outright removed if any issues arise)* |
| 🟨  | **Considering** | The mod is considered to be added.<br/><br/>*This simply means that there is an interest in the mod but is still uncertain on wether to add it due to multiple factors (mainly compatability and if it hinders the overall experience). These are mods that, in short, are not interesting enough to immediately add* |

### Type

| **Icon** | **Type** | **Definition** |
| --- | --- | --- |
| 🟤  | Uncertain | The mod was labeled with any requirements and unsure if it does *(Default value when adding mass mods in the list)* |
| 🟢  | Independent | The mod can run indepent of other mods (excluding the loader itself) |
| 🟣  | Dependency | The mod is required by another mod in order for it to function |
| 🟡  | Dependent | The mod cannot run without the required mod |

### Categories

| Category | Definition |
| --- | --- |
| **Backend** | This is the list of mods that change generation, rendering, logic, etc. of the game in order to optimize the game itself or change the overal game. <br/><br/>*(Library mods that change gameplay should most likely be added here)* |
| **Primary** | This is the list of mods that add a bulk of mods which greatly change the gameplay. <br/><br/>*These are often mods that are heavily prioritized because of interest during the creation of the modpack* |
| **Secondary** | This is the list of mods that add onto the primary mod or are of lower priority. <br/><br/>*these are usually addon mods in a bigger or main mod or are mods that often do add a lot but are not of bigger importance compared to the mods in the primary* |
| **Tertiary** | This is the list of mods that mainly focus on the asthetic side of the mods. <br/><br/>*These are mods that focus on mainly UI changes, animations, etc. It focuses on the overal look rather than adding actual mechanics or gameplay* |
| **Others** | This is the list of mods that do not fit the above categories. <br/><br/>*Uncertain of it's actual categorization. Unsorted. The "Others" pile.* |
